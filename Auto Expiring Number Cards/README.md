# Auto Expiring Number Cards

## What the App Does

- User enters a number
- User clicks the Add button
- A card is created with that number
- The timer starts from the entered number
- The timer decreases every second
- The card disappears when the timer reaches 0

## Main Files

- `src/App.jsx` has the logic
- `src/App.css` has the styling
- `src/main.jsx` starts the app
